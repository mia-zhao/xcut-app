Desplegar una aplicación Next.js en Cloudflare Pages es un proceso sencillo que le permite aprovechar la red global de Cloudflare para un alojamiento rápido y seguro. Aquí hay una guía paso a paso para poner su aplicación Next.js en funcionamiento en Cloudflare Pages.

### Requisitos previos

Antes de comenzar, asegúrese de tener lo siguiente:

- Una aplicación Next.js lista para ser desplegada.
- Una cuenta de Cloudflare. Si no tiene una, puede registrarse de forma gratuita.

### Siga la guía oficial

Siga la guía oficial [aquí](https://developers.cloudflare.com/pages/framework-guides/nextjs/ssr/get-started/).
