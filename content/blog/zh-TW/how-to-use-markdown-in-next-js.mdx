這篇博客是使用 Markdown 撰寫的，展示了如何在您的 Next.js 應用程式中有效地使用 Markdown。Markdown 是一種輕量級的標記語言，允許您輕鬆格式化文本。在這篇文章中，我們將介紹如何在您的 Next.js 項目中設置 Markdown，包括必要的庫、配置和示例圖片。

## 什麼是 Markdown？

Markdown 是一種使用純文本語法格式化文本的簡單方法。它允許您創建標題、列表、鏈接、圖片等，而無需複雜的 HTML 標籤。這使得它成為撰寫內容的絕佳選擇，特別是對於博客和文檔。

### Markdown 的主要特性

- **標題：** 您可以使用 `#` 創建 H1 標題，使用 `##` 創建 H2 標題，依此類推。
- **列表：** Markdown 支持有序和無序列表：
  - 無序列表示例：
    - 無序列表項目 1
    - 無序列表項目 2
  - 有序列表示例：
    1. 有序列表項目 1
    2. 有序列表項目 2
- **鏈接：** 您可以輕鬆創建鏈接：[Next.js 文檔](https://nextjs.org/docs)
- **圖片：** 添加圖片非常簡單：![示例圖片](/logo.svg "這是一個來自公共目錄的示例圖片")

## 在 Next.js 中設置 Markdown

要在您的 Next.js 項目中設置 Markdown，您需要配置您的應用程序以支持 MDX。MDX 允許您在 Markdown 文檔中編寫 JSX，使您能夠在 Markdown 內容中使用 React 組件。

請參考 [官方文檔](https://nextjs.org/docs/app/building-your-application/configuring/mdx) 獲取有關如何在您的 Next.js 項目中設置 MDX 的更詳細說明。
