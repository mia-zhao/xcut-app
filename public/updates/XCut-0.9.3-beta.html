New Features

  • All-New Settings Panel: The entire settings window has been redesigned with a cleaner, more modern interface that is easier to navigate.
  • Check for Updates: XCut now displays version information and can check for updates manually.
  • Floating Icon: A new, optional floating icon provides quick access to the overlay. Enable it in the "Appearance" settings.

Improvements

  • Improved Keyboard Navigation: Navigating the overlay window is now more intuitive. You can seamlessly use the Tab and arrow keys to move between shortcuts and groups.
